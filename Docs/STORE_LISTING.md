# Play Store Listing

## App Name
Breakout+

## Short Description (<= 80 chars)
Neon brick-breaker with 10 modes, rich powerups, and foldable-ready play.

## Full Description
Breakout+ is a modern arcade brick-breaker tuned for smooth high-refresh gameplay on phones and foldables. Build combos, route precision shots, and adapt to distinct game-mode rule sets.

Modes:
- Classic: balanced progression run.
- Timed Challenge: 2:30 score attack.
- Endless: infinite scaling levels.
- God Mode: endless no-life-loss practice.
- Level Rush: 55-second stages with one life.
- Volley: turn-based chain launches with descending rows.
- Tunnel Siege: breach a fortified ring through a narrow gate lane.
- Survival: one life and faster speed scaling.
- Invaders: enemy fleet movement, telegraphed shots, and shield pressure.
- Zen Mode: relaxed flow without score/life pressure.

Powerups (15):
- Multi-ball, Laser, Guardrail, Shield, Extra Life
- Wide Paddle, Shrink, Slow, Overdrive, Fireball
- Magnet, Gravity Well, Ball Splitter, Freeze, Pierce

Brick types (10):
- Normal, Reinforced, Armored, Explosive, Unbreakable
- Moving, Spawning, Phase, Boss, Invader

Highlights:
- Responsive HUD across phone/tablet/foldable sizes
- Smooth aiming and paddle control model
- Daily challenges and per-mode scoreboards
- Distinct visual themes and high-action effects
